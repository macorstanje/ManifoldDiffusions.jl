Geodesics

A geodesic $\gamma:I\to\mathcal{M}$ with $\gamma_0=x\in\mathcal{M}$ and $\dot{γ}_0=v\in T_x\mathcal{M}$ is modelled as a solution to the Hamiltonian system

\[\mathrm{d}x = \nabla_p H(x, p)\mathrm{d}t\]
\[\mathrm{d}p = -\nabla_x H(x,p)\mathrm{d}t\]
ManifoldDiffusions.Geodesic — Function
Geodesic(x₀::Tx, v₀::Tv, tt, ℳ::TM) where {Tx, Tv <: AbstractArray, TM<:EmbeddedManifold}

Returns the values of the geodesic on ℳ starting at x₀ with initial velicity v₀ on a discretized time interval tt. All input is in local coordinates.
